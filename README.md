# ListEverywhere 🏪

AI-powered marketplace listing generator. Take a photo → get listings for 7 marketplaces instantly.

## Deploy to Netlify (free)

### Option A — Drag & Drop (easiest, no GitHub needed)
1. Go to netlify.com and sign up free
2. From your dashboard, drag this entire folder onto the page
3. Netlify builds and deploys it automatically
4. You get a live URL like `yourapp.netlify.app`

### Option B — GitHub
1. Push this folder to a GitHub repo
2. Go to netlify.com → "Add new site" → "Import from Git"
3. Select your repo — Netlify auto-detects the build settings
4. Deploy!

## Add to Phone Home Screen
Once deployed, open the URL on your phone:
- **iPhone**: Safari → Share button → "Add to Home Screen"
- **Android**: Chrome → 3 dots menu → "Add to Home Screen"

It'll work like a real app — full screen, no browser bar!

## How it works
- Upload photos of your item (up to 6)
- Claude AI analyzes the item and writes optimized listings for each marketplace
- Copy any field with one tap, then open the marketplace and paste
- Marketplaces: Facebook, eBay, Depop, Mercari, Poshmark, Craigslist, OfferUp
